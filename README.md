# MERN Note Taking App

A full-stack Note Taking application built with the MERN stack (MongoDB, Express, React, Node.js).

## Features

- **Authentication**: Secure Login and Signup with JWT.
- **Notes Management**: Create, Read, Update, and Delete notes.
- **Search**: Search notes by title or content.
- **Pinned Notes**: Pin important notes to the top.
- **Profile**: Update profile picture.
- **Theme**: Dark and Light mode support.
- **Responsive Design**: Fully responsive UI with a mobile-friendly navbar.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)
